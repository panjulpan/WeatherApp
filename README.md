# WeatherApp

Pre-Test Submission untuk Proses Rekrutmen JDS

Step untuk Menjalankan Aplikasi ini:

- npm install
- npm run android

Jika terjadi error saat melakukan 'npm install' bisa coba di tambahkan flag '--force' atau '--legacy-peer-deps'
